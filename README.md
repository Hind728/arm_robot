# 🦾 Bras Robotique — Pick & Place

Simulation d'un bras robotique 6 axes avec ROS2, Gazebo et MoveIt2, capable de saisir et déposer des objets automatiquement.

## Technologies utilisées

- ROS2 Jazzy
- Gazebo
- ros2_control
- MoveIt2
- C++ / Python


https://github.com/user-attachments/assets/622c1a71-d399-40d4-a956-b1d5fac0d22c
## Lancement

```bash
ros2 launch arm_robot start.launch.py
ros2 run arm_robot pick_and_place
```
